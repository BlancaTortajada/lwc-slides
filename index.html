HTML

  • A semantic language: it says what things are, not what they should look like
  • It's a markup language
  • It works with tags
  • And attributes (classes, ids...)

CSS

  • A style sheet language, it says what things should look like
  • It's linked to the HTML by a link on the head tag
  • It applies a style according to the selectors on the HTML tags

JavaScript

  • It says how things should behave
  • It works on every browser
  • Useful for creating interface interactions
